🖱️ 手動點擊導航測試

這個頁面會開啟實際的 NexusTrade 首頁和市場頁面，測試貨幣卡片點擊導航功能。

📋 測試步驟說明

  1. 點擊下方按鈕開啟首頁或市場頁面
  2. 在新分頁中，找到任意貨幣卡片
  3. 點擊貨幣卡片
  4. 觀察 URL 是否變為 #/currency/SYMBOL 格式
  5. 檢查是否正確載入單一貨幣詳情頁面

🏠 首頁測試

測試首頁熱門貨幣列表的點擊導航

🏠 開啟首頁 (新分頁)
預期行為：
1. 首頁載入完成
2. 熱門貨幣區域顯示
3. 點擊任意貨幣卡片
4. URL 變為 #/currency/BTCUSDT (或其他幣種)
5. 載入單一貨幣詳情頁面

💹 市場頁面測試

測試市場頁面200種貨幣列表的點擊導航

📈 開啟市場頁面 (新分頁)
預期行為：
1. 市場頁面載入完成
2. 200種貨幣列表顯示
3. 點擊任意貨幣卡片
4. URL 變為 #/currency/SYMBOL
5. 載入單一貨幣詳情頁面
